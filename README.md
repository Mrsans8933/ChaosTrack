# 🎵 ChaosTrack

**ChaosTrack** — простой аудиоплеер, который случайным образом выбирает и воспроизводит музыку из указанной папки.  
Идеален для тех, кто устал от выбора треков и хочет довериться случайности.

---

## ✨ Возможности

- 🎲 Случайное воспроизведение треков
- ⏱️ Отображение текущего времени и общей длительности
- 📊 Визуальный прогресс-бар (30 символов)
- 📁 Выбор папки с музыкой через `path.txt`
- ⚠️ Проверка ширины окна терминала (рекомендуется ≥86 символов)
- 🛑 Остановка по `Ctrl+C`
- 🔊 Увиличение/понищения звука по стрелочкам (Прибавить - вверх, убавить - вниз)
- 🚀 Работает без Python (после сборки в `.exe`)
---

## 🚀 Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/Mrsans8933/ChaosTrack
cd ChaosTrack
```
### 2. Установи зависимости
```bash
pip install -r requirements.txt #установка библеотек (зависимостей)
```
### 3. Запусти
```bash
python ChaosTrack.py
```

## Для обычных людей которые хотят все здесь и сейчас (для Windows):
### Просто скачайте ChaosTrack.exe
